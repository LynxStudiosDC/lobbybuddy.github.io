The Command List

  • !skin {skin_name}

    Description: Change the bots skin to the input.

    Example Usage: !skin Nog Ops

  • !backpack {backpack_name}

    Description: Change the bots backpack to the input.

    Example Usage: !backpack Black Shield

  • !pickaxe {pickaxe_name}

    Description: Change the bots pickaxe to the input.

    Example Usage: !pickaxe Raider's Revenge

  • !emote {emote_name}

    Description: Make the bot do an emote based on the input.

    Example Usage: !emote Floss

  • !random {skin/backpack/pickaxe/emote/all}

    Description: Make the bot pick and put on a random cosmetic.

    Example Usage: !random skin

  • !style {cosmetic_name} {variant_type} {variant}

    Description: Make the bot put on a style of a cosmetic.

    Example Usage: !style Skull Trooper clothing_color 2

  • !privacy {public/fiends/private}

    Description: Change the privacy of the party.

    Example Usage: !privacy private
